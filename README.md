✈️Flight Ticket Booking App
A React-based application that fetches flight data from an API and allows users to browse available tickets.

✨ Features
Fetch and display flight data from API
Show flight details (destination, price, time, etc.)
Dynamic and responsive UI
Real-time data rendering

🛠 Built With
React
JavaScript
API (data fetching)

🎯 Purpose
Practice working with APIs, data fetching, and dynamic rendering in React.
